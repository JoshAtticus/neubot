User API

Endpoint

GET /api/user

Headers

  • Cookie: Required (session cookie) for authenticated: true responses.
  • Accept: application/json

Description

Retrieves information about the currently authenticated session.

Response Schema

{
  "authenticated": boolean,      // (Required) top-level status
  "user": {                      // (Optional) Only present if authenticated=true
    "id": string,                // (Required) Unique user ID (e.g. "google_123")
    "name": string,              // (Optional) User's display name
    "email": string,             // (Optional) User's email
    "provider": string,          // (Required) "google" or "github"
    "profile_pic": string        // (Optional) URL to avatar image
  }
}
