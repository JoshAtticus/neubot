Query API

Endpoint

POST /api/query

Description

Processes natural language queries. This is the main entry point for all bot interactions.

Request Body

{
  "query": "what is the weather in Paris?",
  "timezone": "Europe/Paris"
}
  • query (string): The user's natural language input.
  • timezone (string, optional): IANA timezone string (e.g. "America/New_York"). Defaults to server config if omitted.

Response Structure

{
  "response": "The weather in Paris is clear sky...",
  "widgets": [
    {
      "type": "weather",
      "data": { ... }
    }
  ],
  "thoughts": [ ... ],
  "highlightedQuery": "what is the <span...>weather</span>..."
}

Widget Types

The widgets array returns structured data for handling richer responses. Each object has a type and a data payload.

1. Weather Widget

Type: weather

Returned when the user asks for weather info.

{
  "type": "weather",
  "data": {
    "location": "Paris",             // (string) The resolved location name
    "condition": "clear sky",        // (string) Weather condition description
    "temperature": {
      "celsius": 15.2,               // (float) Temperature in Celsius
      "fahrenheit": 59.4             // (float) Temperature in Fahrenheit
    },
    "humidity": 65,                  // (int) Humidity percentage, 0-100
    "description": "The weather..."  // (string) Redundant summary text
  }
}

2. Search Results Widget

Type: search_results

Returned when a Brave Search query is executed.

{
  "type": "search_results",
  "data": {
    "query": "python tutorial",      // (string) The executed search query
    "spellcheck": null,              // (string|null) Suggested spelling correction
    "results": [
      {
        "title": "Welcome to Python",
        "url": "https://python.org",
        "description": "The official home...",
        "favicon": "https://..."
      }
    ],
    "meta": {
      "total": 1,
      "header": "Here's what I found..."
    }
  }
}

3. Home Assistant Widget

Type: home_assistant

Returned when a home automation command is executed.

{
  "type": "home_assistant",
  "data": {
    "summary": "Turned on 1 light.",      // (string) Natural language summary
    "action": "turn_on",                  // (string) The HA service action (turn_on/turn_off)
    "domain": "light",                    // (string) The HA domain (light/switch/script/scene)
    "devices": [                          // (array) List of affected devices
      {
        "entity_id": "light.living_room",
        "name": "Living Room",
        "success": true,                  // (bool) Whether the API call succeeded
        "requested_action": "turn_on",
        "applied_color": "blue",          // (string|null) Color name if applied
        "applied_brightness_pct": 100     // (int|null) Brightness if applied
      }
    ],
    "applied": {                          // (object) Global settings applied to this batch
      "color_name": "blue",
      "brightness_pct": 100
    }
  }
}

4. Fun Result Widget

Type: fun_result

Returned for special easter eggs (e.g. "self destruct", "rainbow mode").

{
  "type": "fun_result",
  "data": {
    "variant": "self_destruct",       // (string) 'self_destruct' or 'rainbow'
    "countdown": 5,                   // (int) Seconds for countdown (if variant=self_destruct)
    "finalText": "BOOM!",             // (string) Text to show after countdown
    "sequence": ["red", "orange..."]  // (array) Colors for rainbow mode
  }
}
