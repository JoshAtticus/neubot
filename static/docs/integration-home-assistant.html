Exposing Home Assistant to the Internet

neubot requires that your Home Assistant instance is externally accessible. Below are the recommended methods.

1. Home Assistant Cloud (Easy)

Requires Payment (Nabu Casa subscription)

1. Open Home Assistant Settings > Home Assistant Cloud.

2. Enable Remote Control.

3. Copy your specific Nabu Casa URL.

Use this URL in the neubot integrations page.

2. Cloudflared Addon (Recommended)

Difficulty: Moderate. Requires free Cloudflare account + Domain.

Requirements

  • Free Cloudflare Account
  • Domain name connected to Cloudflare
  • Home Assistant with Addons support

Steps

  1. Install the Cloudflared addon in Home Assistant.
  2. Configure the addon with your domain (e.g. ha.example.com).
  3. Start the addon and check the logs for the authentication URL.
  4. Visit the URL to authorize the tunnel with Cloudflare.
  5. Add the trusted_proxies config to your configuration.yaml:
http:
  use_x_forwarded_for: true
  trusted_proxies:
    - 172.30.33.0/24

6. Restart Home Assistant.

3. Port Forwarding (Not Recommended)

Caution: Exposing port 8123 directly is a security risk.

If you have a public IP, you can forward port 8123 on your router to your Home Assistant IP.
